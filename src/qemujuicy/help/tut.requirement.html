Prerequisites / Requirements

To run QemuJuicy it is necessary to have QEMU installed (www.qemu.org) on your host system (very often the computer you sit in front of), since QEMU - a pack of applications - contains the simulators for various VMs. In addition, tools like "qemu-img" are used by QemuJuicy to create a virtual "harddisk" of a VM, if the VM is installed on it. The "harddisk" is usually a file on your host system. There is also the possibility to run a Live System from a CD/DVD or an ISO file, which is downloaded or taken from a CD/DVD.

QemuJuicy needs "qemu-img" to create/copy/convert VM harddisks, and one or more emulators for virtual machines. QEMU has emulators for many hardware systems. Each emulator for a specific computer hardware starts with "qemu-system-*", where the "*" describes the hardware (e.g. qemu-system-x86_64 the 64-bit CPUs for AMD and Intel).

QemuJuicy will handle this for you at the first start to guess all necessary applications. If you change your QEMU installation later, you may wish to recheck for this using the File -> Settings menu item, selecting the QEMU installation directory.

Linux / Unix / macOS hosts

Usually, QEMU is already available through your package manager, and after installation located at /usr/bin or on another place of your $PATH variable. If you are interested: "type qemu-img" might show you the directory, "ls /usr/bin/qemu*" lists the installed hardware emulators and some QEMU tools. QemuJuicy will find it automatically in most cases.

Windows hosts

To install QEMU on Windows, download it from www.qemu.org -> Download -> Windows and install it, using the exe or msi file. Most likely, QEMU is installed somewhere in "C:\Program Files\qemu". Mind the space within "Program Files". QemuJuicy will try to find it automatically. To use the QEMU commands on the command line, you might need to add the path to the environment variable "%PATH%".

Other hosts

QEMU comes also as source. Compile it, find the installation directory and choose this directory within the setup wizard to select the QEMU programms for QemuJuicy.

Proof installation if you want, problems

Find the QEMUinstallation directory and choose this directory within the setup wizard to select the QEMU programms for QemuJuicy.

A "qemu-img --version", showing the version on the command line - maybe in the right directory or put in front - gives a proof of a correct installation.
