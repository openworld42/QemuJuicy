QemuJuicy Tutorial

QemuJuicy is Open Source Java application to install and run virtual machines in an easy way, using the QEMU emulator. It has a GUI (graphical user interface) to do so.

QEMU (www.qemu.org) is a software emulator for several hardware machines (processors, CPUs). It uses various hypervisors to emulate hardware.

This tutorial is a help and a description for using QemuJuicy.

In computing, a virtual machine (VM) is the virtualization/emulation of a computer system. Virtual machines are based on computer architectures and provide functionality of a physical computer. Their implementations may involve specialized hardware, software, or a combination (from en.wikipedia.org/wiki/Virtual_machine).

The computer running the virtual machine (QemuJuicy/QEMU) is usually called the "host", the virtual machine (the emulated system) is called "guest" or "client". A host can run many guests.
