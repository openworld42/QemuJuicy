First start

Starting QemuJuicy the first time displays a setup wizard to guide through QemuJuicys setup.

At the first step, QemuJuicy tries to guess all necessary QEMU programms. The directory containing the QEMU installation can be selected, if the QEMU installation directory has not been found automatically. If the automated setup succeeds, the wizard displays "Success" and the version of the QEMU programms.

Second, several general settings of QemuJuicy can be set.

Next, QemuJuicy needs to know where to store its log file and settings. Important is the directory to store the VM harddisk files and properties of the virtual machines. Harddisk file can easily grow to tens of gigabytes, if the installed operating system occupies much space. Depending on the individual backup strategy, one might choose another place to store the VM harddisk files.

Note: the values can be changed later calling the File -> Settings dialog.
